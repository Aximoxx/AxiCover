## Concept
<div align="left">

**Chaque joueur reçoit un mot :**

* Les **Civils** ont le bon mot
* Les **Undercovers** ont un mot proche
* **M. White**, _Giga Chad_, n'a rien du tout.<br>

Le but :  Parler, observer, accuser et voter.
</div>

---

## Rôles

| Rôle           | Objectif                                       |
|----------------|------------------------------------------------|
| **Civil**      | Repérer les imposteurs sans trop en dire.      |
| **Undercover** | Passer pour un Civil avec un mot différent.    |
| **M. White**   | Bluffer sans mot, et deviner le mot avec rien. |

---

## Déroulement
<div align="left">
Une partie, des rôles et des manches qui s'enchaînent.<br>
Chaque tour : vous devez donner un mot proche du vôtre.<br>

À la fin du tour, vote pour éliminer celui qui a donné un mot avec lequel vous n'êtes pas d'accord.
</div>

---
## Fonctionnalité

<div align="left">
Les parties peuvent se jouer avec ou sans chat vocal.<br>
Vous pouvez utiliser Discord ou Mumble.<br>

Un panel de configuration vous permet de setup votre partie comme vous le désirez :
* Point d'apparition (Ajouter, Afficher, Supprimer)
* Nombre d'Undercover (Ajouter, Supprimer)
* Nombre de Tours (Ajouter, Supprimer)
* Type de partie (Vocal ou Textuelle)
* M. White (Activer, Désactiver)
* Sauvegarder la configuration
* et bien plus !
</div>
